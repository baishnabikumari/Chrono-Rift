**CHRONO RIFT** (**New Updates**)
* **see the past and clear present**

## This game is about the dual-reality world platformer built using HTML5 canvas, JS and CSS. You just have to control the character(ball) to clear amazing levels...
## Things are added for the testing purpose.Do not use this to harm someone's privacy or computer this game(website) is made for development and education purpose only.

## UI is full Build Using canvas by me and then used html canvas and css for implementation it into codes.(no AI tools used to make anyting all levels are crafted by me using ASCII text tiles using certain Words Alphabet(like ^,|,D,J,S,E,G(goal) and more) and used 3 to make Blocks)

## New Features 
**Added crumbling Blocks**
* **(--These are the block which fall after the character step on it for 1 sec )**
* **Denoted by C**

**Added Drone**
* **These are the flying shooters, this will act like a firing jet which will fire bullets while the character is in the range of the drone that is (300 Pixels)**
* **Denoted by D**

**Added Springs**
* **These are the spring which make character jump higher than the normal one**
* **Denoted by J**

## Controls
```bash
  ↑(jump), →(forward), ←(backward).

  You can also use! WASD and SPACE for jump as well.

  If you are confused so you may Press the **HOW TO PLAY** inside the game to know about the controls more.
```

## Features
* **Dual Reality(Past & Presenta)**
* **Time Ripple physics**
* **Time buttons** Step on **E** and **press E** to Unlock the secret path(Bridge) which you will see in the present world(right screen).
* **Level switch button for Level Switching(Added more levels and the levels are locked, You have to comeplete the prevoius levels to play the next levels)**
* **Cools sound effects and vibaration effect when user dies or touches the Spikes or Drone**
* **GITHUB button for REPO(for cloning the repo Prefer "How to run" point below this)**

## How to run!
1. **Clone the Repo**
```bash
git clone (Paste REPO-link here(i.e - https://github.com/baishnabikumari/Chrono-Rift/tree/main)).git
```
2. **Navigate the project folder**
```bash
cd Chrono-Rift
```
3. **Run the game**
