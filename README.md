**CHRONO RIFT**
* **see the past and clear present**

## the game is about the dual-reality world platformer built using HTML5 canvas, JS and CSS. You just have to control the character(ball) to clear amazing levels...

## Controls
```bash
  ↑(jump), →(forward), ←(backward).

  You can also use! WASD and SPACE for jump as well.
```

##

## Features
* **Dual Reality**
* **Time Ripple physics**
* **Time buttons** Step on E and press E to Unlock the secret path which you will see in the present world(right screen).
* **Level switch button for Level Switching**
* **Cools sound effects and vibaration effect**
* **GITHUB button for REPO**

## How to run!
1. **Clone the Repo**
```bash
git clone (Paste REPO-link here).git
```

2. **Navigate the project folder**
```bash
cd Chrono-Rift
```
3. **Run the game**
